# lasagna

Layered Zero-Knowledge Proof library for Cardano on-chain scripts.

## Building

```sh
aiken build
```

## Testing

To run tests, simply do:

```sh
aiken check
```

## Documentation

Generate HTML documentation:

```sh
aiken docs
```
